Q2.write a function which takes a string of parenthesis as input and returns if the parenthesis string is properly opened/closed
“(){}” => true
"({)}" => false
"{()}" => true
"[{(})}" => false

Solution:

// // code
// “(){}” => true
// "({)}" => false
// "{()}" => true
// "[{(})}" => false


function check(s) {
    console.log("checking")
    const pairs={
        ")" : "(",
        "}" : "{",
        "]" : "[",
    }
    const stack = []
    for(const str of s)
    {
        if( str === "(" || str === "[" || str === "{")
        {
            stack.push(str)
        }
        else if(str === ")" || str === "}" || str === "]")
        {
            if(stack.length === 0 || stack[stack.length - 1]!== pairs[str])
            {
                return false
            }
            stack.pop()
        }
    }
    return stack.length === 0
    
}
const s = prompt("Enter string")
const isBalancedParenthesis = check(s)
console.log("isBalancedParenthesis:",isBalancedParenthesis)

2. Two sum problem

Given an array of integers and a target value, return indices of the two numbers that add up to the target.

nums = [2, 7, 11, 15], target = 9
Output: [0, 1]

Solution:
 const nums = [2, 7, 11, 15]
const target = 9
const stack = []
for(let i=0;i<nums.length;i++)
{
    for(let j=i+1;j<nums.length;j++)
    {
        if(nums[i] + nums[j] === target) {
        stack.push(i,j)
        }
    }

}
console.log("stack:",stack)

the above solution is not optimal it takes O(n*n)

Optimal solution: O(n)

we need to use hash map
function twoSum(nums,target){
const obj = {}
for(let i=0;i<nums.length;i++) {
const value = target-nums[i]
if(obj[value] != undefined)
{
  return [obj[value],i]
}
obj[nums[i]] = i;
}
return null;
}

const pair = twoSum([2, 7, 11, 15],9)
